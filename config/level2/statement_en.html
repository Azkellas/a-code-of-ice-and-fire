  The Goal

Strategic turn-based game.
You control a faction, starting from an edge of the map.
Each turn, you can:
move units
train units (with a level, 1, 2 or 3)
build something: a mine (that produces more gold), or a tower (that protects nearby cells)
You capture a cell if you walk on it
Each turn, you gain 1 gold per owned cell and 4 per owned mine
You only control/own cells that are connected to your HQ
fight: to kill a unit, yours must be of higher level, or at max level (2 kill 1, 3 kill 3 and 2 and 1, 1 kills nothing)
to kill towers and neighbour cell (4-connex), your unit must be 2 or 3

Win condition: capture the enemy HQ
Have more military power at the end (100 turns)
military power = currentGold + unitsCost for units alive

Each action is performed sequentially

  Game Input

The program must first read the initialization data from standard input. Then, provide to the standard output one line with ...
Input
Initialization input

1 Line: Two ints WIDTH and HEIGHT, the dimensions of the map

Game turn input

First HEIGHT lines:

WIDTH characters: # = void, . = neutral, O = own & active, o = own & inactive, X = active opponent cell, x = inactive opponent cell

Next line: 1 int: buildingCount

Next buildingCount lines:

Four ints: type (0 = HQ, 1 = mine, 2 = tower) owner xy

Next line: 1 int: unitCount

Next unitCount lines:

Five ints: id owner level xy

Output
A single line MOVE id x y
TRAIN level x y
BUILD "MINE"|"TOWER" x y
separated by ;
Example "MOVE 1 2 3; TRAIN 3 3 3; BUILD MINE 0 1"

Constraints
WIDTH, HEIGHT <= 15
... contains between 1 and 500 characters.

Allotted response time to output is ≤ 2 seconds.
